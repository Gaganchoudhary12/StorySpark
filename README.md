# StorySpark Mobile

## Setup

1. Install dependencies: `npm install`
2. Create a `.env` file from `.env.example` and point `API_BASE_URL` to your backend.
3. Start Metro: `npm start`
4. Run on Android/iOS:
   - Android: `npm run android`
   - iOS: `npm run ios`

The app walks through a mood, relationship, and theme selection flow, then generates a story from the backend API.
